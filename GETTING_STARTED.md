# 🚀 빠른 시작 가이드

이현이네와 채이네 가족 게임 플랫폼을 시작하는 가장 빠른 방법입니다!

## 📋 필요한 것

- Node.js 18 이상
- npm 또는 yarn
- Supabase 계정 (무료)
- Google AI Studio 계정 (무료)

## ⚡ 5분 안에 시작하기

### 1단계: Supabase 프로젝트 생성 (2분)

1. https://supabase.com 접속
2. "New Project" 클릭
3. 프로젝트 이름: `family-game`
4. 비밀번호 설정 및 지역 선택 (Seoul)
5. "Create Project" 클릭

### 2단계: 데이터베이스 설정 (1분)

1. Supabase 대시보드에서 "SQL Editor" 클릭
2. "New Query" 클릭
3. `supabase/schema.sql` 파일 내용을 복사해서 붙여넣기
4. "Run" 클릭

### 3단계: API 키 복사 (30초)

1. "Settings" > "API" 메뉴로 이동
2. `Project URL` 복사
3. `anon public` 키 복사

### 4단계: Gemini API 키 발급 (1분)

1. https://makersuite.google.com/app/apikey 접속
2. "Create API Key" 클릭
3. API 키 복사

### 5단계: 프로젝트 설정 (30초)

프로젝트 루트에 `.env.local` 파일 생성:

```env
NEXT_PUBLIC_SUPABASE_URL=여기에_복사한_URL
NEXT_PUBLIC_SUPABASE_ANON_KEY=여기에_복사한_anon_key
GEMINI_API_KEY=여기에_복사한_gemini_key
```

### 6단계: 실행! (1분)

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 http://localhost:3000 열기!

## 🎮 첫 게임 플레이하기

### 회원가입
1. "회원가입" 클릭
2. 닉네임, 이메일, 비밀번호 입력
3. 가입 완료!

### 방 만들기
1. 대기실에서 "새 게임 방 만들기" 클릭
2. 방 이름 입력
3. 게임 유형 선택 (라이어 or 마피아)
4. 최대 인원 설정
5. "방 만들기" 클릭

### 게임 시작
1. 다른 가족들이 방에 입장할 때까지 대기
2. 모두 "준비" 버튼 클릭
3. 방장이 "게임 시작" 클릭
4. 즐기세요! 🎉

## 📱 모바일에서 앱처럼 사용하기

### iPhone (Safari)
1. Safari에서 게임 사이트 열기
2. 하단 공유 버튼 탭
3. "홈 화면에 추가" 선택
4. "추가" 탭

### Android (Chrome)
1. Chrome에서 게임 사이트 열기
2. 메뉴 (⋮) 탭
3. "홈 화면에 추가" 선택
4. "설치" 탭

이제 앱처럼 사용할 수 있습니다!

## 🎯 게임 규칙

### 🎭 라이어 게임
- **목표**: 라이어 찾기 (시민) / 들키지 않기 (라이어)
- **인원**: 3-12명
- **진행**:
  1. 한 명은 라이어, 나머지는 시민
  2. 시민들은 제시어를 받고, 라이어는 카테고리만 받음
  3. 돌아가며 힌트 제시
  4. 투표로 라이어 찾기
  5. 라이어가 잡히면 제시어 맞추기 기회

### 🔪 마피아 게임
- **목표**: 마피아 제거 (시민) / 시민 제거 (마피아)
- **인원**: 4-12명
- **역할**:
  - 마피아: 밤에 시민 제거
  - 시민: 낮에 투표로 마피아 찾기
  - 의사: 밤에 한 명 보호
  - 경찰: 밤에 한 명 조사
- **진행**:
  1. 밤: 마피아 공격, 의사 치료, 경찰 조사
  2. 낮: AI 사회자가 결과 발표
  3. 토론 및 투표
  4. 반복

## ❓ 문제 해결

### 로그인이 안 돼요
- 이메일과 비밀번호를 확인하세요
- 회원가입을 먼저 했는지 확인하세요
- 브라우저를 새로고침하세요

### 실시간 업데이트가 안 돼요
- 인터넷 연결을 확인하세요
- 페이지를 새로고침하세요
- Supabase Realtime이 활성화되었는지 확인하세요

### AI 사회자가 작동하지 않아요 (마피아)
- Gemini API 키가 올바른지 확인하세요
- API 할당량을 확인하세요 (무료: 60 requests/minute)
